# PROJET-OUTIL-DEV_

Ceci est le dépôt lié à ton projet local `Cold War`.

Actions ajoutées par défaut:
- `.gitignore` : règles communes pour OS, IDEs, Node et Python
- `.github/workflows/ci.yml` : workflow CI minimal qui s'exécute sur `push` et `pull_request`

Prochaines étapes recommandées :
- Ajouter un `LICENSE` si tu veux ouvrir le projet
- Ajouter un `.gitignore` plus spécifique si tu utilises un langage/outil particulier
- Configurer une clé SSH pour pousser sans credentials HTTPS

Si tu veux, je peux continuer et appliquer l'une de ces options.
